# SAD-Project
This is the start of the project for building a share price web application, using java, something like yahoo finance and google finance.
The system applies key architectural principles such as Simple Architecture, Clean Architecture and connectors.

## Scope of application
* Gather and keep daily price information on a given share symbol, between two specified dates, up to a
maximum range of two years
* Store the data in a database
* Display graphs of daily share price over time ranges, for one or two different companies, so that
users can view and compare share performance of the companies

## Program Requirements 
* The system should allow users to input a share symbol
* The system should allow users to select a start and end date maximum of 2 years
* The system should retrieve daily share price data from an external data source
* The system should store retrieved data locally in a database
* The system should display share price graphs in order to compare past and present, or between two companies
* The system should allow users to access the stored data if network isnt available

## Program architecture
* The system should follow clean architecture principles
* The system should use layered architecture
* The system should be simple
* The system should be functioning without any errors
